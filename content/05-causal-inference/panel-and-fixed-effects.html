Panel Data & Fixed Effects

  • Show All Code
  • Hide All Code
  • View Source

Watch each unit change against itself over time, and every fixed thing about it — measured or not — cancels out.

The deepest threat to causal claims is the confounder you can’t measure: ability, culture, managerial skill. If you observe the same units repeatedly, a near-magical trick becomes available — compare each unit only to itself across time. Anything about a unit that stays fixed, however unobservable, differences away. Fixed effects buys you control over an entire universe of time-invariant confounders for free — and the fine print is everything it cannot touch.

Why it matters

Matching only handles observed confounders; IV and RD need special variation that isn’t always there. Panel data offers a different escape: by following units over time, you can sweep away all time-invariant unobservables at once. (Li 2023) This is the backbone of the “Furious Five” — difference-in-differences is a two-period fixed- effects design — and the most common way applied researchers defend against unobserved heterogeneity. (Wooldridge 2019)

Panel data and the heterogeneity problem

Panel (longitudinal) data follows the same cross-sectional units — people, firms, countries — across multiple periods. (Li 2023) Model the outcome with a unit-specific term: Y_{it}=\beta_0 + X_{it}\beta + \alpha_i + u_{it}, where \alpha_i is the individual fixed effect — all the unobserved, time-invariant traits of unit i (innate ability in a wage equation, managerial skill in a firm). (Li 2023) The danger: if \alpha_i is correlated with X_{it}, pooled OLS is biased and inconsistent — heterogeneity bias. (Li 2023) (High-ability workers get more schooling and earn more; ignore ability and schooling’s coefficient inherits its effect — the ability bias of Part III, now with a repeated-measures cure.)

The within (fixed-effects) estimator

The fix is disarmingly simple. Demean every variable by its unit-specific mean over time — the within transformation: (Li 2023) (Y_{it}-\bar Y_i) = (X_{it}-\bar X_i)\beta + (u_{it}-\bar u_i). Because \alpha_i is constant over time, its own mean is itself (\bar\alpha_i=\alpha_i), so \alpha_i-\bar\alpha_i=0 — it vanishes. (Li 2023) Every fixed trait, observed or not, is swept away, and \beta is identified purely from within-unit variation over time: when a unit’s X changes, how does its Y change? The figure shows why this matters — pooled and within slopes can point in opposite directions.

Show the figure code 
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(6)
beta_true = -1.0
units, T = 6, 8
colors = plt.cm.viridis(np.linspace(0.1, 0.9, units))

fig, ax = plt.subplots(figsize=(7.2, 4.2))
allX, allY = [], []
for u in range(units):
    alpha = 4.0 * u                          # unit fixed effect, correlated with X below
    x = alpha + rng.normal(0, 0.6, T)        # high-alpha units have high X (confounding)
    y = alpha + beta_true * (x - alpha) + rng.normal(0, 0.3, T)  # within-unit slope is negative
    ax.scatter(x, y, color=colors[u], s=22, zorder=3)
    b = np.polyfit(x, y, 1)                   # within-unit fit
    xs = np.linspace(x.min(), x.max(), 10)
    ax.plot(xs, np.polyval(b, xs), color=colors[u], lw=1.6, alpha=0.8)
    allX.append(x); allY.append(y)

allX = np.concatenate(allX); allY = np.concatenate(allY)
bp = np.polyfit(allX, allY, 1)               # pooled OLS
xs = np.linspace(allX.min(), allX.max(), 20)
ax.plot(xs, np.polyval(bp, xs), "k--", lw=2.4, label=f"pooled OLS slope = {bp[0]:+.2f}")
ax.plot([], [], color="#555", lw=1.6, label=f"within-unit slope ≈ {beta_true:+.2f} (true)")
ax.set_xlabel("X (e.g. schooling)"); ax.set_ylabel("Y (e.g. wage)")
ax.set_title("Fixed effects recovers the within-unit relationship")
ax.legend(fontsize=8, frameon=False, loc="upper left")
fig.tight_layout()
plt.show()

There’s an immediate corollary: a variable that never changes for a unit — race, gender, distance to a fixed river — is demeaned to zero, so fixed effects cannot estimate its effect. (Li 2023) The very trick that kills the confounding also kills any time-invariant regressor of interest.

First differencing, and a cousin

An alternative removes \alpha_i by subtracting consecutive periods — first differencing: \Delta Y_{it}=\Delta X_{it}\beta+\Delta u_{it}. (Li 2023) With exactly two periods, FE and FD are numerically identical. (Li 2023) With more, the choice hinges on the errors: FE is more efficient if the idiosyncratic u_{it} are serially uncorrelated, while FD is better if they behave like a random walk (so differencing whitens them). (Li 2023)

Fixed vs random effects, and the Hausman test

A random-effects (RE) model instead treats \alpha_i as part of a composite error v_{it}=\alpha_i+u_{it}. (Li 2023) The bargain: (Li 2023)

  • RE is more efficient — it uses both within- and between-unit variation and can estimate time-invariant regressors — but only if the RE assumption holds: \operatorname{Cov}(X_{it},\alpha_i)=0, the individual effect is uncorrelated with the regressors.
  • FE is consistent regardless, even when \alpha_i correlates with X — exactly the case that matters for causal work in observational data, where such correlation is the norm.

The Hausman test adjudicates: under the null \operatorname{Cov}(X,\alpha)=0, FE and RE agree; reject the null → use FE (RE is inconsistent); fail to reject → RE for its efficiency. (Li 2023) For causal inference the default leans FE, because the whole reason you reached for panel data was a suspicion that \alpha_i is correlated with treatment.

Two-way fixed effects

Add time fixed effects \lambda_t to absorb shocks hitting all units in a period — a recession, a national policy: (Li 2023) Y_{it}=\alpha_i+\lambda_t+X_{it}\beta+u_{it}. Unit effects sweep away fixed cross-unit differences; time effects sweep away common temporal shocks — and with treatment as the regressor this is difference-in-differences, generalised to many units and periods. (Li 2023) (Which is also why the staggered-adoption critique applies to TWFE here too.)

What fixed effects cannot fix

FE is powerful but not a magic bullet — and knowing its blind spots is the mark of using it well: (Li 2023)

  • Time-varying confounders. FE removes only what’s constant. An unobserved trait that changes and correlates with treatment — a city’s shifting economic climate — still biases the estimate.
  • Reverse causality. If Y also causes X (crime → police spending), FE is biased like any OLS; demeaning doesn’t fix simultaneity.
  • Time-invariant regressors. As noted, their effects are unidentifiable — they difference out.

The honest summary: fixed effects neutralise time-invariant unobserved confounding and nothing else. That single, large class of threats is what they buy — which is why FE is usually a foundation for a design (DiD, event study), not a complete identification strategy on its own.

NoteWooldridge’s job-training grants

Do training grants raise productivity? With a panel of 54 firms, a pure cross-section would be biased — grants may go to firms that are already well-run (high \alpha_i). Fixed effects differences that fixed firm “skill” away, identifying the effect from firms’ changes across years: the 1988 grant is estimated to cut the 1989 scrap rate by ~34.4% (e^{-0.422}-1). (Wooldridge 2019) The effect is only credible because the same firms were watched over time.

Common pitfalls

WarningWhere this goes wrong

  • Believing FE removes all confounding. It removes only time-invariant unobservables; time-varying confounders and reverse causality remain. (Li 2023)

  • Trying to estimate a time-invariant effect with FE. Gender, race, fixed geography difference out — use RE (if its assumption is defensible) or another design. (Li 2023)

  • Defaulting to random effects for efficiency. RE is inconsistent if \alpha_i correlates with X — usually the case in observational data. Run Hausman; lean FE for causal claims. (Li 2023)

  • Ignoring serial correlation. It changes FE-vs-FD efficiency and demands clustered standard errors; un-clustered SEs badly overstate precision in panels. (Li 2023)

  • Too little within-variation. If treatment barely changes within units, FE has almost nothing to work with and estimates are imprecise. (Li 2023)

Appendix · the mathematics

The within transformation kills \alpha_i. Average the model over t for unit i: \bar Y_i=\beta_0+\bar X_i\beta+\alpha_i+\bar u_i (since \bar\alpha_i=\alpha_i). Subtract from the level equation: Y_{it}-\bar Y_i=(X_{it}-\bar X_i)\beta+(u_{it}-\bar u_i), in which \alpha_i and \beta_0 have disappeared. OLS on the demeaned data is the FE estimator \hat\beta_{FE}=\big(\sum_{it}\ddot X_{it}\ddot X_{it}'\big)^{-1}\sum_{it}\ddot X_{it}\ddot Y_{it} with \ddot X_{it}=X_{it}-\bar X_i — a regression on deviations from unit means, so only within-unit covariation identifies \beta.

FE = FD at T=2. With two periods, demeaning gives X_{i2}-\bar X_i=\tfrac12\Delta X_i and likewise for Y; the factor \tfrac12 cancels in the OLS ratio, so \hat\beta_{FE}=\hat\beta_{FD} exactly. This is why the 2×2 DiD can be read as either.

Why Hausman points to FE on rejection. Under H_0:\operatorname{Cov}(X,\alpha)=0, both estimators are consistent but RE is efficient; under H_1 only FE is consistent. The statistic (\hat\beta_{FE}-\hat\beta_{RE})'[\operatorname{Var}(\hat\beta_{FE})-\operatorname{Var}(\hat\beta_{RE})]^{-1}(\hat\beta_{FE}-\hat\beta_{RE}) is \chi^2 under H_0; a large value signals the estimators disagree, indicting RE’s assumption and favouring FE.

Connections

  • Builds on: omitted-variable bias (the unobserved \alpha_i FE removes), multiple regression (FE is OLS on dummies/deviations), and clustering (panel SEs).
  • Leads to: difference-in-differences (two-way FE with a treatment timing) and synthetic control (when you have one treated unit and FE/DiD has no clean control group).
  • Reappears in: virtually all applied field work on administrative panels.

$\ddot X_{it}=X_{it}-\bar X_i$ — a regression on **deviations from unit means**, so only within-unit
covariation identifies $\beta$.

**FE = FD at $T=2$.** With two periods, demeaning gives $X_{i2}-\bar X_i=\tfrac12\Delta X_i$ and
likewise for $Y$; the factor $\tfrac12$ cancels in the OLS ratio, so $\hat\beta_{FE}=\hat\beta_{FD}$
exactly. This is why [the 2×2 DiD](difference-in-differences.qmd) can be read as either.

**Why Hausman points to FE on rejection.** Under $H_0:\operatorname{Cov}(X,\alpha)=0$, both estimators
are consistent but RE is efficient; under $H_1$ only FE is consistent. The statistic
$(\hat\beta_{FE}-\hat\beta_{RE})'[\operatorname{Var}(\hat\beta_{FE})-\operatorname{Var}(\hat\beta_{RE})]^{-1}(\hat\beta_{FE}-\hat\beta_{RE})$
is $\chi^2$ under $H_0$; a large value signals the estimators *disagree*, indicting RE's assumption and
favouring FE.

:::
