Effect Sizes

  • Show All Code
  • Hide All Code
  • View Source

The ‘so what?’ a p-value can’t give you. How big is the effect — in standard deviations, variance explained, or odds — independent of how large your sample happened to be.

A p-value answers “is there an effect?”; it says nothing about how big. And size is what decides whether a result matters — whether a drug is worth taking, a program worth funding. The effect size supplies that magnitude, on a scale that doesn’t inflate just because your sample is large. Cohen’s d, R^2, an odds ratio: each answers the question significance can’t. Reporting it is now the expected standard, not a courtesy.

Why it matters

A p-value is a “measure of surprise,” not of magnitude. (Reinhart 2015) And surprise is cheap when n is huge: a gigantic sample will flag “extremely tiny but relatively unimportant differences” as p<0.05. (Reinhart 2015) The statistical-vs-practical-significance trap is exactly this confusion. Effect size provides the “so what?” — the size of the relationship, independent of sample size. (Reinhart 2015) It is the number a decision actually depends on.

Standardized effect sizes

The most useful effect sizes are standardized, so they compare across studies and instruments: (Reinhart 2015)

  • Cohen’s d — the standardized mean difference between two groups, in pooled-SD units: d = \frac{\bar X_{\text{treatment}} - \bar X_{\text{control}}}{SD_{\text{pooled}}}. The rough conventions: 0.2 small, 0.5 medium, 0.8 large. (Reinhart 2015) A d of 0.8 means the average treated unit is 0.8 SDs above the average control — a gap you’d notice.
  • Correlation r and R^2 — “variance-accounted-for” effect sizes. (Reinhart 2015) R^2 runs 0 to 1 and reports the proportion of variation in the outcome explained, tying directly to regression’s goodness of fit.
  • Odds ratios / risk ratios — for binary outcomes, exponentiating a logistic coefficient gives an odds ratio, the relative change in the odds of the outcome (a hazard ratio plays the same role for durations). (Reinhart 2015)
Show the figure code 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-4, 5, 400)
def normal(x, mu): return np.exp(-0.5*(x-mu)**2) / np.sqrt(2*np.pi)

fig, axes = plt.subplots(1, 3, figsize=(9.6, 3.2), sharey=True)
for ax, d, lab in zip(axes, [0.2, 0.5, 0.8], ["small", "medium", "large"]):
    ax.fill_between(x, normal(x, 0), color="#b7b6a8", alpha=0.7, label="control")
    ax.fill_between(x, normal(x, d), color="#161613", alpha=0.5, label="treatment")
    ax.set_title(f"d = {d}  ({lab})", fontsize=10.5, fontweight="bold")
    ax.set_yticks([]); ax.set_xlabel("outcome (SD units)")
axes[0].legend(fontsize=8, frameon=False, loc="upper right")
fig.suptitle("Cohen's d: the standardized distance between two groups", fontsize=11)
fig.tight_layout()
plt.show()

Statistical vs practical significance, made concrete

The two can point opposite ways: (Reinhart 2015)

  • Significant but trivial. Reinhart’s example: a “banana” that raises SAT scores by 0.9 points is statistically significant in a massive sample and practically meaningless. (Reinhart 2015) Big n + tiny effect = a real-but-pointless result.
  • Large but non-significant. A big effect in a small sample may fail the 0.05 test yet be important and worth following up — the effect size flags it where the p-value buries it. (Reinhart 2015)

This is why the effect size, with its confidence interval, tells the honest story: the CI shows the range of magnitudes the data are consistent with, not just a pass/fail verdict.

The reporting reform

Because p-values are “like mosquitoes” — persistent despite their well-known flaws — journals increasingly require reporting effect sizes and confidence intervals alongside (or instead of) bare significance. (Reinhart 2015) The American Psychological Association has urged this since 1994, so results are read in their “situational context” rather than as a yes/no significance decision. (Reinhart 2015) It’s the same reform Kaplan and the modern critics press: lead with the magnitude.

Common pitfalls

WarningWhere this goes wrong

  • Reporting only the p-value. It hides the magnitude; a reader can’t judge importance without the effect size and its CI. (Reinhart 2015)

  • Reading significance as size. A tiny p in a large sample can mean a trivial effect; check d / R^2. (Reinhart 2015)

  • Dismissing a non-significant large effect. In a small sample it may simply lack power; the effect size says it’s worth a bigger study. (Reinhart 2015)

  • Treating Cohen’s benchmarks as laws. 0.2/0.5/0.8 are rules of thumb; “large” in one field is “small” in another — interpret in context. (Reinhart 2015)
