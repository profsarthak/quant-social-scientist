Cointegration & Error-Correction Models

  • Show All Code
  • Hide All Code
  • View Source

The one case where regressing non-stationary levels is not a crime — when two series share a trend, the long-run relationship between them is real, and the short run is a return to it.

Spurious regression warned: never regress one I(1) series on another. Cointegration is the principled exception. When several non-stationary series are tied together by a common stochastic trend, a particular linear combination of them is stationary — and the levels regression recovers a genuine long-run equilibrium, not a fluke. The companion idea, the error-correction model, says the short run is the system pulling itself back toward that equilibrium whenever it strays.

Why it matters

Many economic relationships are equilibrium statements: consumption and income, spot and futures prices, short and long interest rates. Each series wanders (is I(1)), yet theory says they can’t drift apart forever. Cointegration makes that intuition testable and gives the only legitimate way to estimate a relationship in levels among non-stationary variables (Enders 2014). It is also the bridge from description to dynamics: the error-correction form separates the long-run relationship from the short-run adjustment toward it.

The core idea

Two or more I(1) series are cointegrated if some linear combination of them is stationary (I(0)) (Enders 2014). The series individually meander with no fixed mean, but they share a common stochastic trend, so they never drift arbitrarily far apart — that bounded gap is the long-run equilibrium (Enders 2014). Formally, for a vector \mathbf{x}_t of I(1) variables, cointegration holds if there is a cointegrating vector \boldsymbol{\beta} with \boldsymbol{\beta}'\mathbf{x}_t stationary (Enders 2014).

The textbook picture is the drunk and her dog: both wander unpredictably (two random walks), but a leash keeps the distance between them bounded — that bounded distance is the stationary combination (Enders 2014).

Show the figure code 
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(3)
T = 240
x = np.cumsum(rng.normal(0, 1, T))     # the drunk: a random walk (I(1))

# the leash: a stationary AR(1) deviation
e = np.empty(T); e[0] = 0.0
for t in range(1, T):
    e[t] = 0.75 * e[t-1] + rng.normal(0, 0.8)
y = x + e                              # the dog: follows X, bounded slack -> Y is I(1)
spread = y - x                         # = e, stationary (cointegrating combination)

fig, (axT, axB) = plt.subplots(2, 1, figsize=(7.4, 5.0), sharex=True,
                               gridspec_kw={"height_ratios": [2, 1]})
axT.plot(x, color="#1f4e79", lw=1.6, label="X  (random walk — the drunk)")
axT.plot(y, color="#c0392b", lw=1.6, alpha=0.85, label="Y = X + wobble (the dog)")
axT.set_title("Two I(1) series sharing a common stochastic trend")
axT.set_ylabel("level"); axT.legend(fontsize=8.5, frameon=False, loc="upper left")

axB.plot(spread, color="#3f8f6f", lw=1.4)
axB.axhline(spread.mean(), color="#444", ls="--", lw=1.2)
axB.fill_between(range(T), spread.mean() - 2*spread.std(), spread.mean() + 2*spread.std(),
                 color="#3f8f6f", alpha=0.10)
axB.set_title("Their spread  Y − X  is stationary — the 'leash'")
axB.set_xlabel("time"); axB.set_ylabel("spread")
fig.tight_layout()
plt.show()

Rescuing the spurious regression

Recall that regressing two independent I(1) series gives a fake high R^2 and huge t-stats. If the variables are cointegrated, that regression is no longer spurious — it consistently estimates the meaningful long-run equilibrium toward which the system converges (Enders 2014). Cointegration is precisely the condition that separates a real levels relationship from a coincidence of two wandering trends.

The Engle–Granger two-step method

Engle & Granger (1987) gave a simple recipe (Enders 2014):

  1. Pretest and regress. Confirm every series is I(1), then estimate the long-run equilibrium by OLS in levels: y_t = \beta_0 + \beta_1 z_t + e_t.
  2. Test the residuals. Collect \hat e_t and test it for a unit root with an ADF-type regression \Delta \hat e_t = a_1 \hat e_{t-1} + \sum \beta_i \Delta \hat e_{t-i} + \epsilon_t. If \hat e_t is stationary, the series are cointegrated.
WarningDon’t use ordinary Dickey–Fuller critical values here

Because OLS chose \beta to minimise the residual variance, the residuals are “prejudiced toward looking stationary.” You must use special critical values (Phillips–Ouliaris, or Enders’ Table C), not the standard DF table (Enders 2014).

Weaknesses. Engle–Granger finds only one cointegrating vector even when more exist; it is sensitive to normalisation (which variable goes on the left-hand side can flip the verdict); and it carries small-sample bias (Enders 2014). These are exactly what Johansen’s method fixes.

The error-correction model

The Granger Representation Theorem is the deep result: for I(1) variables, cointegration and error-correction are equivalent representations (Enders 2014). If the series are cointegrated, their short-run changes must respond to the equilibrium error — last period’s gap from the long-run relationship:

\Delta \mathbf{x}_t = \pi_0 + \pi\,\mathbf{x}_{t-1} + \sum_{i=1}^p \pi_i\,\Delta\mathbf{x}_{t-i} + \epsilon_t .

The coefficient \pi (often \alpha) is the speed of adjustment: it governs how quickly the variables return to equilibrium after a shock (Enders 2014). A large |\alpha| means fast correction; \alpha = 0 means that variable doesn’t adjust (it’s weakly exogenous — the long-run forcing variable). The ECM is the best of both worlds: a long-run equilibrium and a short-run dynamic in one estimable equation, all in stationary (\Delta) terms so standard inference works.

The Johansen methodology

Johansen (1988) recast cointegration as a maximum-likelihood, VAR-based problem — sidestepping Engle–Granger’s normalisation issue and allowing multiple cointegrating vectors (Enders 2014). The key object is the rank r of the matrix \pi in the VECM, which counts the cointegrating relations (Enders 2014):

\operatorname{rank}(\pi) Meaning
r = 0 No cointegration (model the differences as a VAR)
0 < r < n r cointegrating vectors (estimate a VECM)
r = n All variables already stationary (a VAR in levels is fine)

Two likelihood-ratio statistics built from the estimated characteristic roots \lambda test the rank (Enders 2014):

  • \lambda_{\text{trace}} — null: at most r cointegrating vectors.
  • \lambda_{\max} — null: exactly r, against r+1.

The system is then estimated as a VECM (vector error-correction model), with \pi factored as \pi = \alpha\boldsymbol{\beta}': \boldsymbol{\beta} holds the cointegrating vectors (the long-run relations) and \alpha the speed-of-adjustment weights (how each equation responds) (Enders 2014).

Common pitfalls

WarningWhere this goes wrong
  • Using standard DF critical values in Engle–Granger step 2. The residuals are biased toward stationarity; use Phillips–Ouliaris/Enders Table C. (Enders 2014)
  • Trusting Engle–Granger with more than two variables. It can miss extra cointegrating vectors and is normalisation-sensitive; use Johansen. (Enders 2014)
  • Running a plain VAR on cointegrated I(1) data. Differencing away a true cointegrating relation loses the long-run information — fit a VECM instead. (Enders 2014)
  • Over-reading a single speed-of-adjustment sign. \alpha must have the right sign and be significant to be “error-correcting”; a wrong-signed \alpha signals a misspecified or non-cointegrated system. (Enders 2014)
  • Claiming cointegration ⇒ causation. A long-run equilibrium constrains the variables; it does not, by itself, establish causal direction. (Enders 2014)

From the sources

NoteWho says what
  • Enders, Applied Econometric Time Series — cointegration as a stationary linear combination of I(1) series sharing a common stochastic trend; the cointegrating vector and the drunk-and-dog leash; how cointegration rescues the levels regression from spuriousness; the Engle–Granger two-step method, the need for special (non-DF) critical values because OLS residuals are “prejudiced toward stationarity,” and its weaknesses (one vector, normalisation-sensitivity, small-sample bias); the Granger Representation Theorem and the error-correction model with the speed-of-adjustment coefficient; and the Johansen ML/VAR approach — the rank of \pi, the \lambda_{\text{trace}} and \lambda_{\max} statistics, and the VECM with \pi=\alpha\beta'.

Appendix · the mathematics

Why \pi = \alpha\beta' encodes cointegration. In the VECM \Delta\mathbf{x}_t = \pi\mathbf{x}_{t-1} + \cdots + \epsilon_t, the term \pi\mathbf{x}_{t-1} must be stationary (everything else is a difference of I(1), hence I(0)). If \pi had full rank n, \mathbf{x}_{t-1} itself would be stationary — contradiction. If \pi=0, there’s no level relationship. The interesting case is reduced rank 0<r<n: then \pi=\alpha\beta' with \alpha,\beta of dimension n\times r, and \beta'\mathbf{x}_{t-1} is the r stationary cointegrating combinations, premultiplied by the adjustment loadings \alpha. The rank of \pi literally counts the long-run relations.

The error-correction intuition in scalars. With long-run relation y_t = \beta z_t and equilibrium error \hat e_{t-1} = y_{t-1} - \beta z_{t-1}, the ECM \Delta y_t = \alpha\,\hat e_{t-1} + (\text{short-run } \Delta\text{ terms}) + \epsilon_t requires \alpha < 0: if y overshot last period (\hat e_{t-1}>0), it is pushed down this period. The half-life of a deviation is \ln(0.5)/\ln(1+\alpha).

Engle–Granger’s “prejudice.” OLS picks \hat\beta to minimise \sum \hat e_t^2; among all linear combinations it finds the most stationary-looking one. Hence even under the null of no cointegration the residuals look “too stationary,” shifting the unit-root test’s distribution left — the reason ordinary DF critical values over-reject and bespoke tables are needed.

Connections

  • Builds on: unit roots (every series must be confirmed I(1) first) and VAR (the VECM is a VAR for cointegrated data).
  • Resolves: the spurious-regression prohibition — cointegration is the licensed exception.
  • Relates to: causal inference — a long-run equilibrium disciplines but does not by itself identify causal direction.

$$

relations [@enders_time_series]:

| $\operatorname{rank}(\pi)$ | Meaning |

| $r = 0$ | **No cointegration** (model the differences as a VAR) |
| $0 < r < n$ | **$r$ cointegrating vectors** (estimate a VECM) |
| $r = n$ | All variables already **stationary** (a VAR in levels is fine) |

- **$\lambda_{\text{trace}}$** — null: at most $r$ cointegrating vectors.

relations) and $\alpha$ the **speed-of-adjustment** weights (how each equation responds)
[@enders_time_series].

  how cointegration **rescues** the levels regression from spuriousness; the **Engle–Granger two-step**
  method, the need for **special (non-DF) critical values** because OLS residuals are "prejudiced toward
  stationarity," and its weaknesses (one vector, normalisation-sensitivity, small-sample bias); the
  **Granger Representation Theorem** and the **error-correction model** with the **speed-of-adjustment**
  coefficient; and the **Johansen** ML/VAR approach — the **rank of $\pi$**, the $\lambda_{\text{trace}}$
  and $\lambda_{\max}$ statistics, and the **VECM** with $\pi=\alpha\beta'$.
:::

## Appendix · the mathematics {.unnumbered}

::: {.math-appendix}

**Why $\pi = \alpha\beta'$ encodes cointegration.** In the VECM
$\Delta\mathbf{x}_t = \pi\mathbf{x}_{t-1} + \cdots + \epsilon_t$, the term $\pi\mathbf{x}_{t-1}$ must be
**stationary** (everything else is a difference of I(1), hence I(0)). If $\pi$ had full rank $n$,
$\mathbf{x}_{t-1}$ itself would be stationary — contradiction. If $\pi=0$, there's no level relationship.
The interesting case is **reduced rank** $0<r<n$: then $\pi=\alpha\beta'$ with $\alpha,\beta$ of dimension
$n\times r$, and $\beta'\mathbf{x}_{t-1}$ is the $r$ stationary cointegrating combinations, premultiplied
by the adjustment loadings $\alpha$. The rank of $\pi$ literally *counts* the long-run relations.

**The error-correction intuition in scalars.** With long-run relation $y_t = \beta z_t$ and equilibrium
error $\hat e_{t-1} = y_{t-1} - \beta z_{t-1}$, the ECM
$\Delta y_t = \alpha\,\hat e_{t-1} + (\text{short-run } \Delta\text{ terms}) + \epsilon_t$ requires
$\alpha < 0$: if $y$ overshot last period ($\hat e_{t-1}>0$), it is pushed **down** this period. The
half-life of a deviation is $\ln(0.5)/\ln(1+\alpha)$.

**Engle–Granger's "prejudice."** OLS picks $\hat\beta$ to minimise $\sum \hat e_t^2$; among all linear
combinations it finds the *most stationary-looking* one. Hence even under the null of no cointegration the
residuals look "too stationary," shifting the unit-root test's distribution left — the reason ordinary DF
critical values over-reject and bespoke tables are needed.

:::
